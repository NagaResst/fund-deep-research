# 如何将深度研究报告转换为网页结构化数据（JSON）

> 本文档说明如何将 `/基金研究报告/` 中的 Markdown 研究报告，手工转换为 `/web-platform/public/data/<code>.json` 文件，供前端网页展示使用。

---

## 核心原则

**不要用脚本自动生成**。报告中有大量判断性文字、定性分析和带背景的数字，需要人工阅读后理解语义，再提炼填入 JSON。  
脚本只适合抓取实时净值等单纯数字，不适合分析性字段。

---

## 一、JSON 文件位置与命名

```
web-platform/public/data/<基金代码>.json     ← 单基金详情
web-platform/public/data/index.json          ← 基金列表（所有基金的摘要）
```

---

## 二、完整字段说明（对应报告章节）

### 顶层结构

```json
{
  "id": "166301",
  "meta": { ... },
  "basic": { ... },
  "fees": { ... },
  "scale": { ... },
  "performance": { ... },
  "risk": { ... },
  "holdings": { ... },
  "managers": { ... },
  "policy": { ... },
  "exclusionCheck": [ ... ],
  "scoring": { ... },
  "navHistory": [],
  "tracking": { ... }
}
```

### `meta` — 报告元数据

来源：报告头部

```json
"meta": {
  "reportDate": "2026-05-08",    // 报告日期
  "dataDate": "2026-05-07",      // 数据截止日期
  "disclaimer": "..."            // 免责声明文本
}
```

### `basic` — 第一章：基础信息表

来源：报告**第一章**信息表格，逐行填入

| JSON 字段 | 报告来源 | 说明 |
|-----------|---------|------|
| `code` | 基金代码 | 字符串 |
| `fullName` | 基金全称 | |
| `shortName` | 基金简称 | |
| `type` | 基金类型（大类） | 混合型/股票型/指数型/债券型 |
| `subType` | 基金类型（子类） | 灵活配置/偏股混合/中证500增强等 |
| `riskLevel` | 风险等级文字 | 低风险/中低/中/中高/高风险 |
| `riskCode` | 风险等级数字 | 1-5 |
| `foundDate` | 成立日期 | YYYY-MM-DD |
| `issueDate` | 发行日期 | YYYY-MM-DD |
| `manager` | 基金管理人全称 | |
| `managerScale` | 管理规模 | 字符串，如 "2029.3亿" |
| `managerFundCount` | 管理基金数量 | 整数 |
| `custodian` | 托管人 | |
| `benchmark` | 业绩比较基准 | |
| `navFallback` | 最新单位净值 | 数字，保留4位小数 |
| `navDateFallback` | 净值日期 | YYYY-MM-DD |
| `inceptionReturn` | 成立以来累计收益率 | 数字，如 1708.9 代表 +1708.9% |
| `dividendCount` | 历史分红次数 | 整数 |
| `status` | 申赎状态 | |
| `companyShort` | 基金公司简称 | 用于展示 |

### `fees` — 费率信息

来源：报告第一章费率行

```json
"fees": {
  "management": 1.2,           // 管理费率，年化 %
  "custodian": 0.2,            // 托管费率，年化 %
  "salesService": 0.0,         // 销售服务费（货币/债券型才有）
  "subscriptionMax": 1.5,      // 最高申购费率 %
  "subscriptionDiscounted": 0.15, // 天天基金优惠后申购费率 %（通常1折）
  "redemptionMax": 1.5,        // 最高赎回费率 %
  "breakdown": [               // 费率明细列表（用于展示）
    { "name": "管理费", "value": 1.2, "note": "年化，从资产中扣除" },
    { "name": "托管费", "value": 0.2, "note": "年化，从资产中扣除" },
    { "name": "申购费", "value": 0.15, "note": "天天基金1折优惠" }
  ]
}
```

### `scale` — 规模数据

来源：报告第一章规模行

```json
"scale": {

  "shares": 3.8037,      // 份额（亿份）

```

### `performance` — 第二章：业绩表现

**`stages`**：2.1 阶段涨幅对比表

```json
"stages": [
  {
    "period": "近1周",    // 时间段名称
    "fund": 4.62,         // 本基金收益率 %
    "peer": null,         // 同类平均（如报告中无数据则 null）
    "hs300": 1.94,        // 沪深300
    "rank": null,         // 同类排名（分子）
    "rankTotal": null,    // 同类总数（分母）
    "quartile": null      // 四分位（"优秀"/"良好"/"一般"/"不佳"）
  }
]
```
> 标准时间段顺序：近1周、近1月、近3月、近6月、今年来、近1年、近2年、近3年

**`annual`**：2.2 历年年度涨幅

```json
"annual": [
  {
    "year": 2018,
    "fund": -14.83,
    "peer": -13.04,
    "hs300": -25.31,
    "rank": 817,
    "rankTotal": 1913,
    "quartile": "良好"
  }
]
```
> 按年份**从旧到新**排列，方便折线图展示

**`quarterly`**：2.3 近期季度表现

```json
"quarterly": [
  {
    "quarter": "2024Q2",   // 格式 YYYYQN
    "fund": -0.99,
    "peer": -2.03,
    "hs300": -2.14,
    "rank": 1116,
    "rankTotal": 2326,
    "quartile": "良好"
  }
]
```
> 按季度**从旧到新**排列（最新季度在最后）

### `risk` — 第三章：风险评估

来源：报告**第三章**

```json
"risk": {
  "volatility": 22.24,       // 年化波动率 %（3.1节）
  "sharpe": 3.5,             // 夏普比率（3.1节）
  "maxDrawdown": -20.5,      // 历史最大回撤 %（负数，报告无则取最差年度亏损）
  "riskLevel": "中高风险",
  "excessReturn1Y": 41.18,   // 近1年超额收益 vs 同类均值 %

  // 4 个指标的注释（展示在指标卡下方）
  "volatilityNote": "...",
  "sharpeNote": "...",
  "drawdownNote": "...",
  "excessNote": "...",

  // 季度仓位变化（3.2节资产配置风险表格）
  "positionHistory": [
    { "date": "2025-03-31", "stock": 68.82, "cash": 31.18, "scale": 53.90 }
    // date: YYYY-MM-DD，stock/cash: %，scale: 净资产亿元
  ],

  // 关键仓位变化信号（如有异常变动则写，否则省略）
  "positionSignal": "⚠️ 2026Q1：基金经理将股票仓位从91.07%主动降至72.27%...",

  // 风险维度分析（3.3节集中度风险 + 综合判断，按风险等级高→低排列）
  "riskBreakdown": [
    {
      "type": "市场波动风险",
      "level": "high",    // "high" / "medium" / "low"
      "stars": 4,         // 风险点数（1-5，越高风险越大）
      "maxStars": 5,
      "note": "..."
    }
  ],

  // 旧字段（兼容用，可忽略）
  "radarDimensions": [ ... ]
}
```

**`riskBreakdown` 来源**：综合第三章各节内容，常见维度：
- 市场波动风险（来自波动率 + 仓位）
- 政策依赖风险（来自政策依赖度）
- 经理变更风险（来自经理任期和履历）
- 行业集中风险（来自行业分布）
- 集中度风险（来自前十大仓位）
- 流动性风险（来自规模和持仓标的）

### `holdings` — 第五章：持仓结构

```json
"holdings": {
  "date": "2026-03-31",        // 季报截止日期
  "stockRatio": 72.27,         // 股票仓位 %
  "cashRatio": 27.64,          // 现金+其他 %
  "top10Total": 24.85,         // 前十大持仓合计 %
  "concentrationLabel": "高度分散型",  // 集中度描述

  // 前十大重仓股（5.1节表格）
  "top10": [
    {
      "rank": 1,
      "code": "601872",
      "name": "招商轮船",
      "ratio": 4.01,           // 持仓比例 %
      "sector": "航运/资源",   // 所属方向（来自报告"所属方向"列）
      "color": "#4492F7"       // 显示颜色（按赛道分配，相同赛道用同色）
    }
  ],

  // 行业分布（5.2节表格，转换为饼图数据）
  "sectors": [
    { "name": "制造业(军工+半导体+新材料)", "value": 56.43, "color": "#4492F7" }
  ],

  // 持仓政策关联（4.2节"持仓与政策关联"表格）
  "policyLinks": [

      "sector": "军工装备",
      "stocks": "洪都航空、航发动力、烽火通信",
      "policyNote": "\"十五五\"强军战略，国防预算年增7%+",
      "color": "#ef4444"
    }
  ]
}
```

**持仓颜色分配原则**（同一基金内统一）：
- 军工：`#ef4444`
- 半导体/科技：`#8b5cf6`
- AI/算力/光模块：`#06b6d4`
- 新能源：`#2DC78E`
- 新材料：`#D4A017`
- 资源/大宗/航运：`#f59e0b`
- 消费/医疗：`#4492F7`
- 金融：`#9B72F0`
- 指数成分（增强/联接）：按指数特征选色

### `managers` — 第六章：基金经理评估

```json
"managers": {
  "current": {
    "name": "童立",
    "startDate": "2025-03-12",
    "tenure": "1年57天",
    "tenureReturn": 72.97,         // 本基金任期收益 %
    "peerAvgReturn": 35.39,        // 同期同类平均 %
    "rankInPeer": 987,             // 任期同类排名（分子）
    "rankTotal": 8154,             // 同类总数（分母）
    "education": "经济学硕士",
    "experience": "约15年（2011年加入华商基金）",
    "fundCount": 9,                // 当前管理基金数
    "totalScale": 83.54,           // 当前管理总规模（亿元）

    // 历任基金完整战绩（6.1节表格）
    "historicalFunds": [
      {
        "name": "华商研究精选A",
        "code": "004423",
        "tenure": "8年140天",
        "return": 353.77,          // 任期收益 %
        "peerAvg": 106.75,
        "alpha": 247.02,           // 超额收益 = return - peerAvg
        "rank": 63,
        "rankTotal": 1977,
        "percentile": 3.2,         // 百分位 = rank/rankTotal*100，保留1位
        "grade": "good",           // "excellent"(前5%) / "good"(前20%) / "average" / "poor"
        "isCurrent": true          // 是否为当前在管基金
      }
    ]
  },

  // 历任经理汇总（6.3节表格，按时间从旧到新）
  "history": [
    {
      "name": "田明圣",
      "start": "2012-09-06",
      "end": "2013-12-29",
      "tenure": "1年114天",
      "return": 11.60,
      "note": "初创期"
    }
  ]
}
```

**`grade` 判断标准**：
- `excellent`：同类前 5%（percentile ≤ 5）
- `good`：同类前 20%（5 < percentile ≤ 20）
- `average`：同类前 50%（20 < percentile ≤ 50）
- `poor`：同类后 50%（percentile > 50）

### `policy` — 第七章：政策匹配度与预测

```json
"policy": {
  // 标签云（当前持仓政策共振主题）
  "tags": [
    { "label": "军工装备", "strength": "high", "color": "#ef4444" }
    // strength: "high" / "medium" / "low"
  ],

  // 三情景分析（7.4节表格）
  "scenarios": [
    {
      "type": "悲观",          // "悲观" / "基准" / "乐观"
      "color": "#ef4444",      // 悲观红、基准蓝、乐观绿（固定）
      "probability": 20,       // 概率 %（三者之和应为100）
      "trigger": "...",        // 触发条件（来自报告"触发条件"列）
      "returnLow": -25,        // 预期收益下限 %
      "returnHigh": -15        // 预期收益上限 %
    }
  ],

  // 短期政策顺风分析（7.1节，每个板块一条）
  "policyBreakdown": [
    {
      "sector": "军工装备",
      "color": "#ef4444",
      "icon": "🛡️",
      "horizon": "短期（1年内）",
      "logic": "...",           // 政策逻辑说明
      "stocks": ["洪都航空（军机）", "航发动力（发动机）"]
    }
  ],

  // 十五五规划方向（7.2节，通常3条）
  "fifteenFive": [
    {
      "direction": "高端装备制造",
      "color": "#ef4444",
      "description": "...",     // 规划方向描述
      "holdings": "洪都航空、航发动力"  // 持仓中对应的标的
    }
  ],

  // 长期风险（7.3节表格，按风险等级高→低排列）
  "longTermRisks": [
    {
      "risk": "基金经理风险",
      "signal": "...",          // 预警信号描述
      "level": "high"           // "high" / "medium" / "low"
    }
  ],

  // 历史阶段双栏时间轴（第四章复盘，每个阶段一条）
  "dualTimeline": [
    {
      "period": "2022",
      "manager": {
        "event": "神操作：逆涨+13.6%，同类第9/2299",
        "return": "+13.60%",
        "note": "...",
        "type": "highlight"   // 见下方颜色说明
      },
      "market": {
        "event": "美联储激进加息，A股全面熊市",
        "note": "...",
        "type": "bear"        // 见下方颜色说明
      }
    }
  ]
}
```

**`dualTimeline` 颜色类型**：

左侧（基金经理/基金表现）：
- `positive`：红色 — 涨幅跑赢、超额收益
- `highlight`：黄色 — 特殊操作（主动降仓、神操作、逆市防守等值得标注的行为）
- `warning`：绿色 — 跌幅/落后/阶段失误
- `neutral`：灰色 — 过渡期、建仓期

右侧（市场/政策背景）：
- `bull` / `policy`：红色 — 市场有利、政策顺风
- `neutral`：黄色 — 市场不确定、震荡分化
- `bear`：绿色 — 市场负面、熊市、逆境

> **注意**：报告中描述的同一年份，左右颜色可以相反（如2022年：左=黄特殊操作，右=绿熊市）

### `exclusionCheck` — 第八章：排除法检查

来源：报告**第八章**表格，10项一票否决规则

```json
"exclusionCheck": [
  {
    "item": "基金公司黑名单",
    "pass": true,
    "note": "华商基金未列入黑名单"
  }
]
```

固定 10 项（顺序按报告）：
1. 基金公司黑名单
2. 基金经理黑名单
3. 规模过小（<1亿）
4. 成立不足1年
5. 基金经理频繁更换
6. 历史最差年跌幅>50%
7. 单一重仓>30%
8. 行业集中度>70%
9. 近1年排名后50%
10. 费率结构异常

### `scoring` — 第九章：投资建议

来源：报告**第九章**

```json
"scoring": {
  "total": 86,   // 综合评分（满分100），来自9.1节综合评分行

  // 6个维度评分（9.1节表格）
  "dimensions": [
    { "name": "历史业绩", "score": 9.5, "note": "..." }
    // 标准维度：历史业绩/风险控制/持仓质量/基金经理/政策匹配/估值时机
  ],

  "recommendation": "观望等待回调，分批建仓",   // 9.2节建议标题
  "recommendationType": "cautious",             // "buy" / "cautious" / "sell"

  // 5条理由（9.2节"理由"列表）
  "reasoning": [
    { "positive": true,  "text": "..." },   // positive=true → ✅正面
    { "positive": false, "text": "..." }    // positive=false → ⚠️风险
  ],

  // 报告结论总结（报告末尾"一句话总结"）
  "conclusion": "...",

  // 适合人群（9.3节）
  "suitableFor": [
    { "label": "投资期限3年以上", "fit": true },
    { "label": "短期流动性需求强", "fit": false }
  ],

  // 配置建议（9.4节表格）
  "allocationSuggestions": [
    { "scenario": "中长期底仓（3年+）", "ratio": "5%-15%", "note": "..." }
  ]
}
```

**`recommendationType` 判断**：
- `buy`：综合评分 ≥ 83 且当前时机合适（未处于明显高位）
- `cautious`：评分良好但当前时机需谨慎（近期大涨/高位/经理刚换）
- `sell`：有一票否决项触发或评分 < 60

## 三、index.json 字段说明

`index.json` 是基金列表页的数据源，每只基金一条摘要：

```json
[
  {
    "id": "166301",
    "shortName": "华商新趋势优选灵活配置混合",
    "type": "混合型",
    "subType": "灵活配置",
    "riskLevel": "中高风险",
    "riskCode": 4,
    "return1Y": 79.24,      // 近1年收益率 %（来自 performance.stages）
    "return3Y": 84.94,      // 近3年收益率 %（<3年基金填 null）
    "score": 86,            // 综合评分（来自 scoring.total）
    "recommendation": "观望等待回调，分批建仓",
    "recommendationType": "cautious",
    "navFallback": 18.089,           // 最新净值（来自 basic.navFallback）
    "reportDate": "2026-05-08",      // 报告日期（来自 meta.reportDate）
    "manager": "童立"                // 现任经理姓名（来自 managers.current.name）
  }
]
```

## 四、操作流程

1. **阅读全文**：完整阅读研究报告，重点关注各章节结论性文字
2. **从 166301.json 复制模板**：以现有完整文件为模板，清空数值，逐章填写
3. **按章节顺序填写**：
   - 第一章 → `basic` + `fees` + `scale`
   - 第二章 → `performance`（stages / annual / quarterly）
   - 第三章 → `risk`（指标 + riskBreakdown + positionHistory）
   - 第四章 → `policy.dualTimeline`（历史复盘，每个关键阶段一条）
   - 第五章 → `holdings`（top10 + sectors + policyLinks）
   - 第六章 → `managers`（current + history）
   - 第七章 → `policy`（tags + scenarios + policyBreakdown + fifteenFive + longTermRisks）
   - 第八章 → `exclusionCheck`（10项）
   - 第九章 → `scoring`（total + dimensions + recommendation + reasoning + conclusion + suitableFor + allocationSuggestions）
   - 第十章 → `tracking`（weekly + quarterly + alerts）
4. **验证 JSON**：运行 `python3 -c "import json; json.load(open('public/data/<code>.json')); print('OK')"`
5. **更新 index.json**：在列表中追加该基金的摘要条目
6. **注意事项**：
   - JSON 字符串内禁止使用英文双引号 `"`，需换为 `「」` 或转义为 `\"`
   - 所有数字字段不加引号（`1.2` 不是 `"1.2"`），`null` 不加引号
   - `pass` 是保留字，JSON 中用 `"pass": true` 没问题（JSON 无保留字限制）
