# 基金深度研究 Skill（完整版）

## 🎯 核心目标

输入基金代码 → **多步骤数据获取** → **强制完整性检查** → **联网搜索补充** → 生成高质量研究报告

**关键原则**：
1. ✅ 脚本是起点，不是终点
2. ✅ 数据缺失必须联网搜索补充
3. ✅ 报告不能有过多N/A字段
4. ✅ 所有数据必须是最新的

### Step 4：📊 数据验证与交叉核对

**在生成报告前，必须验证**：

1. **数据一致性检查**
   - 基金名称在各处是否一致
   - 净值数据是否合理（不会突然暴涨暴跌）
   - 规模数据是否在合理范围

2. **时效性检查**
   - 净值日期是否是最近的（T-1或当日）
   - 季报数据是否是最新的
   - 基金经理是否现任

3. **合理性检查**
   - 收益率是否合理（债券基金年化一般3%-8%，股票基金-30%到+50%）
   - 夏普比率是否合理（一般0-3之间）
   - 最大回撤是否符合基金类型

**发现问题时**：
- 重新搜索验证
- 标注数据疑点
- 在报告中说明

**报告必须包含以下章节**（缺一不可）：

- 成立时间、规模

#### 4. 历史表现与市场/政策关联分析（核心章节·深度复盘）

**必须深入分析基金历史涨跌与市场环境、政策变化的关联性**：

##### A. 关键时间节点复盘
**识别基金历史上的重要涨跌节点，并分析原因**：

1. **大幅上涨阶段**（涨幅>20%的时期）
   - 时间段：XXXX年XX月 - XXXX年XX月
   - 累计涨幅：XX%
   - **驱动因素分析**：
     - 市场因素：牛市/熊市？流动性宽松？风险偏好提升？
     - 政策因素：哪些政策支持了持仓行业？
     - 行业因素：持仓行业景气度如何变化？
     - 基金经理操作：是否主动调整仓位或行业配置？
   
2. **大幅下跌阶段**（跌幅>15%的时期）
   - 时间段：XXXX年XX月 - XXXX年XX月
   - 累计跌幅：XX%
   - **驱动因素分析**：
     - 市场因素：系统性风险？流动性收紧？
     - 政策因素：监管收紧？补贴退坡？贸易摩擦？
     - 行业因素：行业周期下行？竞争加剧？
     - 基金经理操作：是否及时减仓或调仓？

3. **横盘震荡阶段**
   - 时间段和表现
   - 原因分析：市场观望？政策真空期？行业调整？

**搜索关键词示例**：
- "{基金名称} {基金代码} 历史净值走势 分析"
- "{基金名称} {年份}年 业绩回顾"
- "{持仓行业} {年份} 政策影响 市场分析"

##### B. 与市场指数的对比分析
**将基金表现与相关指数进行对比**：

1. **基准指数选择**
   - 股票型基金：沪深300、中证500、创业板指
   - 债券型基金：中债综合指数、10年期国债收益率
   - 混合型基金：偏股混合型基金指数
   - 行业主题基金：对应行业指数（如新能源指数、半导体指数）

2. **超额收益来源分解**
   - **择时能力**：基金经理是否在市场低点加仓、高点减仓？
   - **选股能力**：重仓股是否跑赢行业平均？
   - **行业配置**：行业轮动是否成功？
   - **风格因子**：价值/成长、大盘/小盘风格暴露

3. **不同市场环境下的表现**
   - **牛市期间**：是否跟上或超越指数？
   - **熊市期间**：是否有效控制回撤？
   - **震荡市期间**：是否有稳定的超额收益？
   - **风格切换期间**：是否及时调整风格？

**量化指标**：
- Beta值（相对于基准指数的敏感度）
- Alpha值（超额收益）
- 信息比率（风险调整后超额收益）
- 跟踪误差（与基准指数的偏离度）

##### C. 重大政策事件影响分析（⚠️ 必须精确到文号、量化目标、净值传导幅度）

**【政策周期分段框架】**

首先将基金历史划分为若干「政策周期段」，每段必须包含：

| 字段 | 要求 |
|------|------|
| 阶段名称 | 红利期 / 调整期 / 修复期 / 政策真空期 |
| 时间区间 | XXXX-MM 至 XXXX-MM |
| 核心政策 | 政策全称 + 文号（如发改能源〔2017〕1701号） |
| 政策量化目标 | 如"到2025年新型储能装机≥3000万千瓦" |
| 区间净值涨跌幅 | XX% |
| 同期沪深300 | XX% |
| 超额收益 | XX% |

**必须输出「政策-净值对照量化表」**（每个完整周期一行）：

| 时间段 | 主要政策（含文号） | 政策核心量化目标 | 基金净值增长率 | 同期沪深300 | 超额收益 |
|--------|-------------------|-----------------|--------------|------------|----------|
| 2017年 | 《储能指导意见》发改能源〔2017〕1701号 | 十三五期间储能商业化初期 | XX% | XX% | XX% |
| ... | ... | ... | ... | ... | ... |

**1. 正面政策案例（每个至少包含以下字段）**
- **政策全称 + 文号 + 发布部门 + 日期**
- **政策核心量化目标**（装机目标/补贴金额/渗透率目标等）
- **对持仓行业的传导路径**：政策→行业需求/供给→重仓股利润→基金净值
- **净值反应量化**：政策发布后 1个月 / 3个月 / 6个月 的涨跌幅
- **基金经理应对**：是否在政策发布前后调整仓位

**2. 负面政策案例（每个至少包含以下字段）**
- **政策全称 + 文号 + 发布部门 + 日期**
- **负面内容摘要**（补贴退坡幅度/监管要求/禁止事项等）
- **对持仓的冲击路径**
- **净值反应量化**：政策发布后各阶段跌幅
- **基金经理是否及时减仓**（通过持仓变化判断）

**3. 最近6个月最新政策专项分析（必须单独成段）**
- 列出最近6个月内发布的所有相关政策
- 按以下维度分类：供给侧（规范竞争/质量监管）/ 需求侧（补贴/目标/采购）/ 技术侧（研发补贴/首台套奖励）
- 分析每条政策对基金重仓股的直接影响
- 给出「政策红利传导路径」：政策→受益细分领域→基金重仓股→预期净值弹性

**4. 政策敏感度评估**
- 敏感程度（高/中/低）+ 量化依据（政策发布后平均反应幅度）
- 政策传导时间滞后（即时/数周/数月），并举例说明
- 历史政策驱动型涨跌占总涨跌幅的比例估算

**搜索关键词示例**：
- "{持仓行业} 历年政策变化 影响分析"
- "{基金名称} 政策敏感度"
- "中国 {年份} 重大经济政策 对市场影响"

##### D. 市场周期适应性评估
**评估基金在不同市场周期中的适应能力**：

1. **经济周期**
   - 复苏期表现（通常利好周期股、金融股）
   - 过热期表现（通常利好资源股）
   - 滞胀期表现（通常利好防御性板块）
   - 衰退期表现（通常利好债券、现金）

2. **流动性周期**
   - 宽松期表现（利好成长股、高估值股票）
   - 收紧期表现（利好价值股、低估值股票）

3. **风险偏好周期**
   - 风险偏好上升期（利好小盘股、题材股）
   - 风险偏好下降期（利好大盘股、蓝筹股）

**评估结论**：
- 该基金最适合的市场环境是什么？
- 在哪些市场环境下表现较差？
- 基金经理是否有逆周期操作能力？

##### E. 归因分析总结
**综合以上分析，给出归因结论**：

1. **历史收益的主要来源**
   - XX%来自市场Beta（跟随大盘）
   - XX%来自行业配置Alpha
   - XX%来自个股选择Alpha
   - XX%来自择时操作

2. **基金经理的核心能力**
   - 擅长什么？（选股/择时/行业轮动/风险控制）
   - 不擅长什么？（需要规避的市场环境）

3. **可复制性评估**
   - 历史业绩有多少可以归因于运气（市场红利）？
   - 有多少可以归因于能力（可持续的Alpha）？
   - 未来能否复制历史业绩？

##### A. 政策趋势预测（未来1-3-5年）

- **短期（1年）**：基于已搜索的最新政策，分析直接影响

- **中期（3年）**：政策演变路径预测
  - "十五五"规划各阶段实施节奏（必须提及具体量化目标）
  - 产业政策可能的调整方向
  - 国际形势对国内政策的影响（如贸易摩擦、技术封锁）

- **长期（5年+）**：结构性变化预测
  - 国家战略方向的长期影响
  - 技术革命带来的政策变革
  - 人口结构、气候变化等宏观因素

##### B. 行业景气度预测
- **持仓行业未来3年景气度**：
  - 行业生命周期阶段（萌芽/成长/成熟/衰退）
  - 供需格局变化预测
  - 竞争格局演变
  - 技术创新带来的颠覆风险
  
- **量化预测指标**：
  - 行业增速预测（vs GDP增速）
  - 利润率变化趋势
  - 估值水平合理性判断

##### C. 情景分析与概率评估
**必须构建至少3种情景**：

1. **基准情景（概率50-60%）**
   - 政策按预期推进
   - 经济平稳增长
   - 基金表现预测

2. **乐观情景（概率20-30%）**
   - 政策超预期支持
   - 经济强劲复苏
   - 基金表现上修

3. **悲观情景（概率10-20%）**
   - 政策不及预期或转向
   - 经济下行压力
   - 基金表现下修

**每种情景需要**：
- 触发条件
- 发生概率（基于历史数据和当前形势）
- 对基金净值的影响幅度
- 应对策略建议

##### D. 风险预警信号
**列出未来需要重点监控的信号**：
- 政策转向信号（如监管收紧、补贴退坡）
- 行业拐点信号（如产能过剩、需求萎缩）
- 宏观经济信号（如利率变化、汇率波动）
- 黑天鹅事件预案

##### E. 动态调整建议
**基于预测给出操作建议**：
- 什么情况下应该加仓？
- 什么情况下应该减仓？
- 什么情况下应该清仓？
- 再平衡触发条件

#### 7. 排除法检查
- 10项一票否决规则检查结果

#### 8. **投资建议（必须有明确结论）**
- 综合评分（0-100分）
- 明确建议：强烈推荐/推荐/观望/谨慎/放弃
- 适合人群
- 配置建议（比例、期限）
- 风险提示

#### 9. 后续跟踪计划
- 日常跟踪要点
- 季度复盘内容
- 预警信号

## 💡 常见问题处理

### Q1: 脚本全部返回N/A怎么办？
**A**: 
1. 不要慌，这是正常的（API有限制）
2. 立即进行联网搜索
3. 从基金名称开始，逐步补充所有信息
4. 参考案例：009520中欧鼎利债券C

### Q2: 搜索结果冲突怎么办？
**A**:
1. 优先采用权威来源（天天基金、晨星、基金公司官网）
2. 优先采用最新数据
3. 在报告中标注数据来源
4. 如有重大分歧，列出多个版本供用户判断

### Q3: 找不到某些数据怎么办？
**A**:
1. 尝试不同关键词组合
2. 查看基金定期报告（季报、年报）
3. 使用基金F10页面
4. 如果确实找不到，标注"暂无公开数据"，不要编造

### Q4: 用户催促快速出结果怎么办？
**A**:
1. 坚持质量标准，不能牺牲完整性
2. 可以先给出简要结论，再补充详细报告
3. 告知用户："为了保证报告质量，需要X分钟完成完整分析"
